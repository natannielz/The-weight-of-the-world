# Nier: Automata - Weight of the World Review

Immersive scrollytelling experience.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. **Important Audio Setup**:
   - Download the instrumental version of "Weight of the World" (The End of YoRHa Version).
   - Name the file `weight_of_the_world.mp3`.
   - Place it in the `public/audio/` folder:
     `public/audio/weight_of_the_world.mp3`

3. Run the development server:
   ```bash
   npm run dev
   ```

## Tech Stack
- React 19
- Tailwind CSS 4
- React Three Fiber / Three.js
- Framer Motion
